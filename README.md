# 🌍 EarthDAO

> Funding the Planet, Together.

EarthDAO is an open-source decentralized crowdfunding platform that connects people with environmental and social impact projects around the world.

## 🚀 Vision

To build the world's most transparent platform where anyone can discover, fund, and track projects that create a positive impact on our planet.

## ✨ Current Features

- 🔐 Secure user authentication
- 👤 User profiles
- 📢 Project creation
- 🌱 Community-driven platform
- ☁️ Supabase backend
- ⚡ Next.js frontend
- 📱 Mobile-friendly interface

## 🛠 Technology Stack

- Next.js
- React
- TypeScript
- Supabase
- PostgreSQL
- Netlify
